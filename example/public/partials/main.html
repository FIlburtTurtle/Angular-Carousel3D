Angular Carousel3D

An AngularJS carousel directive in 3D using HTML5 3D transformations. Meant to be lightweight and flexible for many projects. Just thought that the current 3D carousels were to rigid in the way they were built!

Getting Started

Git clone, download the project at https://github.com/FIlburtTurtle/Angular-Carousel3D, or bower install angular-carousel3d. Include angular-carousel3d.css and angular-carousel3d.js as shown below.

<link rel="stylesheet" href="stylesheets/Angular-Carousel3D.css" /> <script src="bower_components/angular/angular.min.js"></script> <script src="js/Angular-Carousel3D.js"></script>

When instantiating your angular module include Carousel3D as a dependency to use the directive.

var example = angular.module('example', ['Carousel3D']); example.controller...

Using the directive

Include a carousel3d element in your project with some of the configuration options.

<!-- 3d Carousel directive --> <!-- Options --> <!-- Rotate : exposes the directive's rotate method --> <!-- Rotateto : rotates to a specific index in the panels array --> <!-- toggleAxis : the toggle axis method to toggle the carousels axis --> <!-- percentHeight : set the max height of the panels (0-1) --> <carousel3d rotate="rotate" rotateto="rotateto" panels="panels" toggle-axis="toggleAxis" toggle-backface="toggleBackface" percent-height='.50'> </carousel3d> <!-- template for 3d carousel directive --> <!-- create a script template with the id of panel3d.html to create a customized panel for the template --> <script type="text/ng-template" id="panel3d.html"> <div class="view" style="padding:20px"> <h2 style="color:white" >pane number {{panel}}</h2> </div> </script> <!-- the panel object is derived from the panels attr you pass to the carousel3d directive -->
