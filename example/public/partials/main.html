Angular Carousel3D

A simple 3D AngularJS carousel directive in using 3D transformations based on the work done by https://github.com/desandro/3dtransforms. Meant to be lightweight and flexible for projects where WebGL is not needed. Just thought that the current 3D carousels were to rigid in the way they were built, this directive allows you to create the template you use within the carousel.

Getting Started

Git clone, download the project at https://github.com/FIlburtTurtle/Angular-Carousel3D, or bower install angular-carousel3d. Include angular-carousel3d.css and angular-carousel3d.js as shown below.

<link rel="stylesheet" href="stylesheets/Angular-Carousel3D.css" /> <script src="bower_components/angular/angular.min.js"></script> <script src="js/Angular-Carousel3D.js"></script>

When instantiating your angular module include Carousel3D as a dependency to use the directive.

var example = angular.module('example', ['Carousel3D']); example.controller('mainController', ['$scope', function($scope){ $scope.panels = [1,2,3,4,5,6,7] $scope.addPanel = function(){ $scope.panels.push($scope.panels.length + 1) } $scope.popPanel = function(){ $scope.panels.pop() } }])

Using the directive

Include a carousel3d element in your project with some of the configuration options.

<!-- 3d Carousel directive --> <!-- Options --> <!-- Rotate : exposes the directive's rotate method --> <!-- Rotateto : rotates to a specific index in the panels array --> <!-- toggleAxis : the toggle axis method to toggle the carousels axis --> <!-- percentHeight : set the max height of the panels (0-1) --> <carousel3d rotate="rotate" rotateto="rotateto" panels="panels" toggle-axis="toggleAxis" toggle-backface="toggleBackface" percent-height='.50'> </carousel3d> <!-- template for 3d carousel directive --> <!-- create a script template with the id of panel3d.html to create a customized panel for the template --> <script type="text/ng-template" id="panel3d.html"> <div class="view" style="padding:20px"> <h2 style="color:white" >panel number {{panel}}</h2> </div> </script> <!-- the panel object is derived from the panels attr you pass to the carousel3d directive -->

Directive attributes


rotate(positions_to_rotate) :
Exposes the rotate function in the directive to the controller, it takes an integer that will rotate the carousel a number of panels. Eg. -1 will rotate back one and 20 will rotate 20 slides forward

rotateto(position_index) :
Exposes the rotateto function to the controller, takes an integer that rotates to the specified index in the panels array.

toggleAxis :
Exposes the function which rotates the carousel's axis

percentHeight :
determines the height of each individual slide, default if 80% of innerHeight

panels :
the array that you pass to the directive through scope, this panels array is watched by the directive and will change when the panels array changes in the controller

actOnAllPanels(key, value) :
Utility function to alter all of the objects in the panels array. It will set all objects with a key and a value.


Custom panels


Include a ng-template with an id of panel3d.html and it will be included at the template for the carousel.
