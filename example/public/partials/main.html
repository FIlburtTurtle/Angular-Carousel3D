Angular Carousel3D

An AngularJS carousel directive in 3D using HTML5 3D transformations. Meant to be lightweight and flexible for many projects. Just thought that the current 3D carousels were to rigid in the way they were built!

Try if out by using some of its functionality below





Getting Started

Please git clone or download the project at https://github.com/FIlburtTurtle/Angular-Carousel3D. Include the two scripts, in the dist folder, angular-carousel3d.css and angular-carousel3d.js after your angularjs include. When instantiating your angular module include Carousel3D as a dependency to use the directive.
